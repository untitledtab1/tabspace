О проекте

Главная цель проекта - показать наши умения, попрактиковаться, но наша команда решила реализовать что-то по-настоящему полезное для геймера. Здесь вы сможете увидеть некоторые новости и гайды игрового мира, которые были актуальны на момент создания.

О разработке

Проект был написан на HTML с подключением CSS. Для разработки использовался простой блокнот, чтобы показать наши навыки написания сайтов даже без использования профессиональных сред разработки.
